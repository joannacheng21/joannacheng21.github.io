Introduction/Background

The swift detection of crop diseases is essential to maximizing global food supply. However, manual detection of plant diseases is inefficient; it requires a high level of expertise and can be highly error prone [3]. This has spurred research to use machine learning techniques, like random forest classifiers, to identify crop diseases in order to prevent the spread of harmful blights [2]. Other papers have ventured beyond traditional machine learning methods; one group used CNNs to classify a dataset of diseased and healthy plant leaves to achieve a 99.3% accuracy [1]. We plan to use the PlantVillage dataset, which contains 54,303 images of healthy and diseased plant leaves from 13 species. It can be found here.

Problem Definition

We aim to develop a machine learning model that can differentiate between whether a given leaf image is healthy or unhealthy across a variety of species and diseases.

Preprocessing Methods

Contrast Enhancement

Utilize scikit-image library (the rescale_intensity method) to increase the contrast in the plant images, which would make it easier for our CNN model to identify the plants from the background.

Smoothing Edges

Utilize gaussian function within the skimage.filters module to smooth edges and remove noise from the leaf images. Using the gaussian function on the plant images would remove any unnecessary features that don’t correlate with disease in the plant leaves, helping our algorithms perform better.

Grayscale

Utilize rgb2gray method from the skimage.color module to convert the colored plant leaf images to grayscale. Training our model with both colored as well as grayscale images would help the model identify disease utilizing the color of the leaf as well as other important features such as lesions.

Algorithms/Models

Convolutional Neural Network

Train a Convolutional Neural Network (CNN) through TensorFlow by using Conv2D and MaxPooling2D layers. First, the image features will be extracted with convolutional layers. Then, with the FC layers and feature information, the model will conclude whether the leaf is from a healthy or unhealthy plant.

Decision Tree Classifier

Use DecisionTreeClassifier class from Scikit to classify leaf data into classes of being healthy or unhealthy. Scikit used the CART (Classification and Regression Trees) algorithm to back its decision tree classifier, which diverges from the C4.5 algorithm in that it doesn’t compute rule sets.

Bagging Classifier

Use sklearn.ensemble.BaggingClassifier to classify our data. This classifier works by forming predictions based on randomized subsets of data; then by combining the predictions from the randomized subsets, the classifier forms a final prediction. By randomizing, this classifier is useful since it reduces variance, helping reduce overfitting.

Quantitative Metrics

  • Accuracy: proportion of correctly identified diseased plants
  • Precision: True Positives/(True Positives + False Positives)
  • Recall: True Positives/(True Positives + False Negatives)
  • F1 Score: Harmonic mean of precision and recall

Project Goals

Our goal is to develop an effective model to identify diseased and healthy plants. We aim to achieve a classification accuracy, precision, recall rate, and F1 Score above 0.90.
